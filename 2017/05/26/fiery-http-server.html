2017-05-26

前文说到使用 opencpu 来搭建 http 服务，opencpu 可以很快速的通过构建 R 包的方式来搭建 http 服务，
很快捷，而且支持各种响应机制。但我们在搭建线上服务时，经常有需求将请求响应的时间控制在100ms以内，opencpu的框架就存在问题了。
这里再介绍R的另外一个包：fiery，部署更加方便且响应优势更加明显（一般30ms以内）。
